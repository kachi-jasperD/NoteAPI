# Notes API

A simple Notes API built with Node.js.

## Getting Started

Follow the steps below to run the project locally.

### 1. Clone the Repository

```bash
git clone <repository-url>
```

### 2. Navigate to the Project Directory

```bash
cd <project-folder>
```

### 3. Install Dependencies

```bash
npm install
```

### 4. Start the Development Server

```bash
npm run dev
```

The application will start in development mode. Once the server is running, you can begin testing the API using your preferred API client (e.g., Postman or Insomnia).

## Available Scripts

- `npm install` – Installs all project dependencies.
- `npm run dev` – Starts the application in development mode.
